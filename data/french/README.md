# Ken French 49 Industry Portfolios (Daily)

Source: Ken French Data Library
(https://mba.tuck.dartmouth.edu/pages/faculty/ken.french/data_library.html),
file `49_Industry_Portfolios_daily_CSV.zip`, built from the 2026-05 CRSP
database. Downloaded 2026-07-14. Value- and equal-weighted daily returns;
missing values coded -99.99/-999. Used by `code/e_french*.py` under the
pre-registrations in `PREREG_FRENCH49.md`.
